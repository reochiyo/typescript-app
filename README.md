# TYPESCRIPT-APP

## Description
This app created by Typescript.

## Requirements
- Typescript 5.4.5
- React 18.2.66
- Vite.js 5.2.0
- Express.js 4.19.2

## Installation
Install dependencies : 
```
$ npm install 
```

## Usage
Set up frontend(port:5173):
```
$ npm run dev
```
Set up backend(port:3001):
```
$ npm run dev
```
